# 🎓 University Event Management System

## 📌 Description

This project is a **University Event Management System** designed to help manage and organize campus events efficiently. It allows users to create events, register participants, and manage event details in a structured way.

---

## 🚀 Features

* Create and manage events
* Student registration for events
* View upcoming and past events
* Manage participant details
* Feedback and review system

---

## 🛠️ Technologies Used

* Frontend: React / HTML / CSS
* Backend: Node.js / Express (or Java if your project is Java-based)
* Database: MongoDB / File System

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/kasuni-17/University-Event-Management-System.git
```

### 2. Navigate to project folder

```
cd University-Event-Management-System
```

### 3. Install dependencies

```
npm install
```

### 4. Run the project

```
npm start
```
